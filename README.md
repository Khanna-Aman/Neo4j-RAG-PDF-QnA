# PDF RAG Application - Phase 1

A React-Node.js application for PDF processing with Neo4j graph database and RAG capabilities.

## Phase 1: Project Setup and Environment Configuration

### Prerequisites
1. **Node.js LTS** - Download from [nodejs.org](https://nodejs.org/)
2. **Neo4j Desktop** - Download from [neo4j.com](https://neo4j.com/download/)
3. **Windows Firewall** - Allow port 7687 for Neo4j
